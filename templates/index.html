Apa itu Reflected XSS?

Reflected XSS terjadi ketika aplikasi menerima data dalam HTTP request dan menyertakan data tersebut dalam respons secara tidak aman. Script berbahaya "dipantulkan" kembali ke browser korban.

Dampak Serangan

  • Pencurian Session/Cookie.
  • Defacement (Mengubah tampilan web).
  • Phishing lewat manipulasi form.
  • Redirection ke situs berbahaya.

Strategi Mitigasi

Gunakan Context-Aware Encoding. Jangan pernah merender input user langsung ke HTML. Gunakan filter modern dan Content Security Policy (CSP).
